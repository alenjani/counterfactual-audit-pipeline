# Reviews

This folder collects independent reviews of the CAP repository against the
ISR paper plan and the broader four-paper research program.

## Conventions

- One file per review.
- Filename: `NNN-YYYY-MM-DD-short-slug.md` where `NNN` is a zero-padded
  monotonically increasing review number (`001`, `002`, …) and the slug is
  a brief description (e.g. `mvp-review`, `pre-l4-readiness`).
- Each review's frontmatter captures: review number, date, reviewer,
  reviewed commit hash, and scope.
- Reviews are append-only history. Don't edit prior reviews to "fix" them
  retroactively — file a new review that supersedes or amends.
